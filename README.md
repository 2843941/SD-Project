#SD-Project
SA Learnerships and Skills Development Portal

#PROJECT DESCRIPTION
This project is a web application designed to manage learnerships and skills development opportunities in South Africa. It allows applicants to create profiles, upload CVs, and apply for opportunities posted by providers. Admins can approve listings to ensure compliance and credibility. The system also integrates SAQA/NQF data for qualifications.

#PRODUCT BACKLOG (Features Draft)
Main features of the Web-app:

Applicant profiles (with CV upload, skills, qualifications)
Opportunity listings (posted by providers)
Application workflow (apply, shortlist, reject, offer)
Notifications (status updates)
Admin approval of listings
SAQA/NQF integration for qualifications

#SPRINT 1 USER STORIES
Who–What–Why format:

1. As an Applicant, I want to create a profile so that I can apply for opportunities.
2. As a Provider, I want to post a learnership so that applicants can see it.
3. As an Applicant, I want to upload my CV so that providers can review my qualifications.
4. As an Admin, I want to approve listings so that only valid opportunities are visible.
